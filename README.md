An end-to-end data science project analyzing SpaceX Falcon 9 launches using
API data collection, exploratory data analysis, and geospatial visualization,
interactive dashboards, and machine learning.

## Objective
Predict Falcon 9 first-stage landing success and identify key factors
influencing launch outcomes.

## Tools & Technologies
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- SQL (SQLite)
- Folium (Geospatial Analysis)
- Plotly Dash (Interactive Dashboard)
- Scikit-learn (ML Models)
- GitHub

## Key Analysis
- Launch site, payload mass, orbit type, and booster version analysis
- Interactive dashboard for real-time analytics
- Proximity analysis of launch sites
- Classification models with hyperparameter tuning

## Model Performance
- **Best Model:** Logistic Regression
- **Accuracy:** **94.44%**
